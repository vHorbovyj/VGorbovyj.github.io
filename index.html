Біографія

У 2014 році отримав диплом молодшого спеціаліста у БАТІ за фахом «Обслуговування комп’ютерних систем і мереж».

Суміщаючи навчання та роботу, у період із 2013 по 2014 працював також над версткою газети → «БережІнфо», дизайном оголошень, листівок, меню, рекламних банерів для закладів харчування та магазинів. Цей період був плідним в освоєнні Adobe Photoshop, Adobe Illustrator, Adobe Indesign, CorelDraw.

З липня 2014 по серпень 2016 років - перебував у складі добровольчого формування, створеного для захисту державного суверенітету, територіальної цілісності та незалежності України, і брав безпосередню участь в антитерористичній операції. Отримав статус учасника бойових дій.

У відпустках між ротаціями вивчав HTML i CSS, пройшов онлайн-курс "Основи Web UI розробки" від Prometheus. → сертифікат

У 2016 вступив на другий курс НТУ «ХПІ» і почав навчання за спеціальністю "Геометричне моделювання та графічні інформаційні технології".

Протягом першого семестру ознайомився із програмою КОМПАС-3D. → курсова робота
